# Systems Engineering Projects

Welcome to the Systems Engineering projects repository for the Politecnic University of Aguascalientes. This repository contains various projects and resources for students enrolled in the Systems Engineering program.

## How to Contribute
For projects created to collaborate on classes: 
1. Fork the repository.
2. Create a new branch (`git checkout -b feature-branch`).
3. Make your changes.
4. Commit your changes (`git commit -m 'Add some feature'`).
5. Push to the branch (`git push origin feature-branch`).
6. Open a pull request.
